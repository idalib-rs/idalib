# Skills for contributing to idalib-rs projects

NOTE: the guidelines in `idalib-rust-style` should be followed by all contributors--whether using AI or not.

## Installation

Via `claude`:

```
/plugin marketplace add idalib-rs/idalib
/plugin install idalib-rust-skills@idalib-rs
```

Then:

```
/idalib-rust-style
```
